To run it in Github :

Inputs :
  GitHub secrets :
                AWS_SECRET_ACCESS_KEY
                AWS_ACCESS_KEY_ID
  For a user that has the following roles :
    IAMFullAccess
    CloudWatchLogsFullAccess
    AWSLambda_FullAccess
